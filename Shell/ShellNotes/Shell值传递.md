
```
echo "Shell 传递参数实例！";
echo "执行的文件名：$0";
echo "第一个参数为：$1";
echo "第二个参数为：$2";
echo "第三个参数为：$3";

执行：
./shell.sh 3 4 5
```

参数说明：

1. `$#` : 传递到脚本的参数个数
2. `$*` : 以一个单字符串显示所有向脚本传递的参数
3. `$$` : 脚本运行的当前进程的ID号
4. `$@` : 与$*相同，但是使用时加引号，并在引号中返回每个参数
5. `$-` : 显示Shell使用的当前选项
6. `$?` : 显示最后命令的退出状态。0->无错误


```
for i in "$*"; do
	echo $i
done

> 3 4

for i in "$@";do
	echo $i
done
> 3
  4

```
